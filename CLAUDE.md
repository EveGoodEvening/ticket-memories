# CLAUDE.md

## Lessons

- Ticket Memories MVP should be language-architecture extensible, but the first UI may be single-language.
- Ticket Memories should cover concerts, music festivals, exhibitions, travel, and sports rather than narrowing only to concerts.
- Prioritize a beautiful timeline and annual recap experience; use `.pkpass` import and manual creation as the reliable MVP ingestion paths.
- MVP sharing should export images/PDFs only; do not build public share URLs initially.
- Physical keepsakes should start as print-ready PDFs before full physical order fulfillment.
- Do not promise automatic reading of all Apple Wallet passes; Wallet access must be framed as user-chosen import.
